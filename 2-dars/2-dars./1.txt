AI (Sun’iy intellekt)
   └── Machine Learning (Ma’lumotlardan o‘rganish)
          └── Deep Learning (Neyron tarmoqlar orqali o‘rganish)

ML turlari:

1. Supervised Learning (Nazoratli o‘qitish)
-Ma’lumotda kirish  va chiqish  mavjud.
-Model input → outputni o‘rganadi.
-Misol: uy narxi (X = maydon, joylashuv; Y = narx).

2.Unsupervised Learning (Nazoratsiz o‘qitish)
-Faqat input ma’lumot bor, output yo‘q.
-Model o‘xshashliklarni topadi (clustering).
-Misol: mijozlarni guruhlash (marketing uchun).

3.Reinforcement Learning (Kuchaytirib o‘qitish)
Agent muhitda harakat qiladi, mukofot yoki jazoga qarab o‘rganadi.
Misol: shaxmat, robotni boshqarish, o‘yin o‘ynash.
